# GameRpg API

Серверная часть RPG-приложения, реализованная на **ASP.NET Core Web API**.  
Проект предоставляет REST API для мобильного клиента на **.NET MAUI** и реализует бизнес-логику игры: аутентификацию, персонажей, навыки, инвентарь, торговлю, друзей, поддержку и административные функции.

---

## Возможности

- Регистрация, авторизация и смена пароля.
- Создание и управление персонажем.
- Просмотр и развитие навыков.
- Работа с инвентарём и экипировкой.
- Торговля между игроками через лоты.
- Система друзей и заявок.
- Сообщения в поддержку.
- Административное управление пользователями.
- JWT-аутентификация и авторизация.

## Команды 

# Контейнер с БД (Postgres)
```dockerfile
cd Api/
docker compose up -d
```

# Запуск сервера
```commandline
dotnet restore
dotnet build
dotnet run --project Api/
```
